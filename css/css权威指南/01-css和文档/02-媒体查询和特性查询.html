媒体查询：通过媒体描述符，根据指定媒体类型的特性选择样式表

可以在以下地方使用：
1. link元素的media属性
2. style元素的media属性
3. @import声明的媒体描述符部分
4. @media声明的媒体描述符部分

简单媒体查询：

@media all{ h1{color:red} } all：代表所有媒体类型。print：为有视力的用户打印文档时使用，也在预览打印时使用。screen：在屏幕媒体上使用。

媒体描述符：包含一个媒体类型和一个或多个媒体特性列表，特性描述符要放在括号里面。 @media all and (min-resolution: 96dpi) {}
逻辑关键词：
1.and：连接两个或多个媒体特性必须同时满足条件，(color) and (orientation:landscape)
2.not：只能在媒体查询开头使用，对整个查询取反，not (color) and (orientation:landscape)

特性描述符：包含 width,min-width,max-width,color,height,min-height,max-height等

特性查询：根据用户是否支持特定的css属性及其值来应用一段样式。特性查询也支持逻辑操作。

@supports (display:grid){ section#main{ display:grid; } }
